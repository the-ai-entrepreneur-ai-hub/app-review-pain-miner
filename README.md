# App Review Pain Miner

Extract AI-powered product intelligence from App Store and Google Play reviews. Identify user pain points, feature requests, and sentiment patterns automatically.

## Features

- Scrape reviews from Apple App Store and Google Play Store
- AI-powered sentiment analysis and categorization
- Automatic pain point and feature request extraction
- Competitive app comparison and benchmarking
- Trend detection across review timelines
- Export structured insights in JSON or CSV

## Use Cases

- Product roadmap prioritization from user feedback
- Competitive intelligence on rival apps
- Identify underserved user needs for new features
- Monitor app reputation and review trends
- Market research for app development

```json
{
  "appUrls": ["https://apps.apple.com/app/id123456789"],
  "maxReviews": 1000,
  "analyzeSentiment": true,
  "extractPainPoints": true
}
```
